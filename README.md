# 🎹 여기 있어 피아노 (Easy Piano)

어린이와 초보자를 위한 웹 브라우저 피아노입니다. 설치 없이 브라우저만 있으면
바로 연주할 수 있으며, **기본 · 코드 · 발성** 세 가지 모드를 제공합니다.

> Created by [교육뮤지컬 꿈꾸는 치수쌤](https://litt.ly/chichiboo)

---

## ✨ 주요 기능

앱을 실행하면 먼저 **모드 선택 화면**이 나타나고, 원하는 모드를 골라 시작할 수 있습니다.
모드에 들어가면 화면 **좌측 상단의 `모드 선택`** 버튼으로 언제든 다시 돌아올 수 있어요.

화면 **우측 상단의 설정 토글**로 **다크 모드(어둡게/밝게)** 와 **건반 계이름 표기 켜기/끄기** 를
선택할 수 있으며, 선택은 브라우저에 저장되어 다음에 열 때도 유지됩니다.

기본 · 발성 모드 화면은 **스크롤 없이 한 화면에 담기도록** 반응형으로 구성되어 있고,
제작자 정보(푸터)는 항상 페이지 최하단에 고정됩니다.

### 1. 기본 모드 🎹
- 1옥타브 / 2옥타브 건반을 자유롭게 연주합니다.
- 마우스, 터치, 멀티터치(여러 손가락 동시 연주)를 지원합니다.
- **서스테인 페달 버튼**: 켜 두면 누른 음이 길게(약 3.5초) 울려 퍼집니다.
  소리가 끝나기 전에도 같은 건반을 다시 눌러 이어서 연주할 수 있습니다.
- **녹음 기능**: 연주를 녹음하여 이름을 붙여 저장하고, 다시 듣기 · 다운로드 · 이름 변경 · 삭제할 수 있습니다.
  (녹음 목록은 브라우저 로컬 스토리지에 저장됩니다.)
- **메트로놈**: 박자(4/4 · 3/4)와 BPM(직접 숫자 입력, 40~220)을 정해 일정한 박을 들려줍니다.
  첫 박에 강세를 주어 박자를 느끼며 연습할 수 있습니다.

### 2. 코드 모드 🎼
- 메이저 / 마이너 / 세븐(7) / 메이저7 / 마이너7 / 디미니시 / 서스포(sus4) 등
  **49개의 코드 버튼**(7개 루트 × 7개 코드 종류)을 제공합니다.
- 코드 버튼을 누르면 해당 화음이 소리 나고, **피아노에서 눌린 건반이 함께 표시**됩니다.
- **현재 코드 표시**: 누른 코드의 이름과 구성음(계이름)을 화면 위쪽에 보여줍니다.
  (예: `Am7 → 라 · 도 · 미 · 솔`)
- **서스테인 페달 버튼**: 켜 두면 누른 음이 일정 시간(약 3.5초) 동안 길게 울려 퍼집니다.
  코드 버튼뿐 아니라 직접 누른 건반에도 적용됩니다.

#### 🎶 자동 반주 (Auto-Accompaniment)
단순히 코드를 확인하는 것을 넘어, **노래 · 리코더 · 뮤지컬 넘버 연습을 위한 간단한 자동 반주 피아노**로
사용할 수 있습니다. 코드 버튼을 눌러 진행을 만들고, 원하는 스타일·템포로 자동 연주시켜 보세요.

- **코드 진행 입력**: 코드 버튼을 누르면 소리가 나는 동시에 **코드 진행에 추가**됩니다.
- **한 마디 안에 여러 코드 (마디 쪼개기)**: `코드 길이`를 **한 마디 / ½ / ¼ (박 단위)** 로 정해
  한 마디에 코드를 1~4개까지 넣을 수 있습니다.
  (예: `C  D`처럼 한 마디를 둘로 나누거나, `G F E`처럼 1박–1박–2박으로 넣어 `1-2-3-3` 리듬 구현)
- **네 마디 단위 코드 차트**: 입력한 코드가 악보처럼 **마디줄(|)과 함께 네 마디 단위**로 표시되어
  진행을 한눈에 확인할 수 있습니다. 연주 중인 마디·코드가 실시간으로 강조됩니다.
- **편집 모드(✎)**: 연필 버튼을 누르면 각 코드에 **삭제(×) 버튼**이 나타나고,
  **드래그로 순서를 바꿀 수 있습니다.** (평소에는 삭제 버튼을 숨겨 깔끔하게 유지)
- **반주 스타일 6종 + 스타일 섞기**
  - **기본**: 코드를 동시에 한 번 재생 (현재까지의 기본 동작)
  - **아르페지오**: 코드음을 오르락내리락 펼쳐서 재생
  - **발라드**: 왼손 베이스 위에 오른손 코드음이 차례로 쌓이는 잔잔한 반주
  - **8비트**: 가요 반주처럼 리듬감 있는 코드 스탭 + 베이스
  - **왈츠**: 쿵–짝–짝 3박자 반주 (3/4 박자에 잘 맞습니다)
  - **뮤지컬**: 낮은 베이스를 깔고 펼침화음과 높은 음으로 극적인 느낌의 반주
  - 두 번째 선택칸에서 다른 스타일을 골라 **두 스타일을 겹쳐(섞어)** 연주할 수 있습니다.
- **재생 / 정지 / 반복(ON·OFF)**: 입력한 코드 진행을 스타일·템포에 맞춰 순서대로 연주합니다.
- **반주 녹음**: 자동 반주를 녹음해 **기본 모드와 동일한 방식**으로 저장·재생·다운로드합니다.
  (녹음 목록은 기본 모드와 공유됩니다.)
- **코드 진행 내보내기**: 입력한 코드 진행을 **이미지(PNG)** 또는 **텍스트(TXT)** 파일로 저장하거나,
  **클립보드로 복사**할 수 있습니다. (가사 없는 코드 차트 형태)
- **템포(BPM)**: 슬라이더 또는 **숫자 입력칸을 직접 클릭해 수정**(60~160, 기본 90)할 수 있습니다.
- **박자 선택**: **4/4 · 3/4 · 6/8** 지원. (왈츠는 3/4, 6/8은 8분음표 6박 흐름에 맞춰 동작)
- **카운트인(ON·OFF)**: 자동 반주 시작 전에 1마디 카운트인을 넣을 수 있습니다.
  (4/4 → `1 2 3 4`, 3/4 → `1 2 3`, 6/8 → `1~6` · 메트로놈 클릭 + 화면 카운트 표시)

### 3. 발성 모드 🎤
한 패턴을 **조(C → D → E → … → B)를 옮겨 가며** 반복하는 **자동 발성 연습** 모드입니다.
조가 바뀌어도 각 조의 메이저 음계를 사용하므로 자연스럽게 음역을 넓혀 줍니다.

- **음 개수(패턴) 선택**
  - **5음 — 아르페지오(1-3-5-3-1도)**: 도-미-솔-미-도
    예) C: 도-미-솔-미-도 · D: 레-파#-라-파#-레 · F: 파-라-도-라-파
  - **9음 — 순차 진행(1~5도 왕복)**: 도-레-미-파-솔-파-미-레-도
    예) F: 파-솔-라-시♭-도-시♭-라-솔-파 · A: 라-시-도#-레-미-레-도#-시-라
- **방향 선택**
  - **오르락내리락**: 낮은 조 → 높은 조로 올라갔다가 다시 내려옵니다. (반복)
  - **오르막길**: 올라가는 방향만 반복합니다.
  - **내리막길**: 내려가는 방향만 반복합니다.
- **시작 조 선택**: C · D · E · F 중에서 발성을 시작할 조를 고를 수 있습니다.
  (선택한 조부터 B까지 차례로 진행)
- **속도 조절**: 슬라이더로 느리게 ↔ 빠르게 조정할 수 있습니다.
- **키 코드 먼저 듣기(코드 안내)**: 켜 두면 각 조로 넘어갈 때 그 조의 **메이저 코드(도·미·솔)를 한 번 들려준 뒤**
  발성을 시작합니다. 조가 바뀌었다는 것을 귀로 확인하며 음을 잡을 수 있습니다. (ON·OFF 가능)
- **재생 / 멈춤**: 연주 도중 언제든 멈출 수 있습니다.
- 연주되는 음에 맞춰 피아노 건반과 음계 가이드가 실시간으로 강조됩니다.
  가이드 계이름은 이동 도법(movable-do)으로 항상 `도-미-솔…` / `도-레-미…`를 유지합니다.

> **오픈소스에 감사드립니다 🙏** — 이 앱은 위 오픈소스 프로젝트들 덕분에 만들어졌습니다.
> 코드 진행 차트 이미지는 별도 라이브러리 없이 브라우저 내장 **Canvas API**로 직접 그리며,

> 모바일에서도 일관된 사용 경험을 제공합니다.

## 🚀 실행 방법

웹 폰트와 오디오 샘플을 불러오기 위해 **인터넷 연결을 권장**합니다.
피아노 샘플(Salamander) 로딩에 실패하거나 시간이 오래 걸리면, 자동으로
**Tone.js 내장 신디사이저 음색으로 전환**되어 끊김 없이 바로 연주할 수 있습니다.

### 바로 열기
`index.html` 파일을 웹 브라우저로 열면 됩니다.

### 로컬 서버로 실행 (권장)
일부 브라우저는 `file://` 환경에서 오디오 정책이 제한될 수 있어 간단한 로컬 서버 사용을 권장합니다.

```bash
# Python 3
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

### GitHub Pages 배포
저장소 설정의 **Settings → Pages** 에서 브랜치를 선택하면 `index.html`이 그대로 정적 페이지로 배포됩니다.
