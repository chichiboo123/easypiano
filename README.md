# 🎹 여기 있어 피아노 (Easy Piano)

어린이와 초보자를 위한 웹 브라우저 피아노입니다. 설치 없이 브라우저만 있으면
바로 연주할 수 있으며, **기본 · 코드 · 발성** 세 가지 모드를 제공합니다.

> Created by [교육뮤지컬 꿈꾸는 치수쌤](https://litt.ly/chichiboo)

---

## ✨ 주요 기능

앱을 실행하면 먼저 **모드 선택 화면**이 나타나고, 원하는 모드를 골라 시작할 수 있습니다.
언제든 좌측 상단의 **`모드 선택`** 버튼으로 다시 돌아올 수 있어요.

### 1. 기본 모드 🎹
- 1옥타브 / 2옥타브 건반을 자유롭게 연주합니다.
- 마우스, 터치, 멀티터치(여러 손가락 동시 연주)를 지원합니다.
- **서스테인 페달 버튼**: 켜 두면 누른 음이 길게(약 3.5초) 울려 퍼집니다.
- **녹음 기능**: 연주를 녹음하여 이름을 붙여 저장하고, 다시 듣기 · 다운로드 · 이름 변경 · 삭제할 수 있습니다.
  (녹음 목록은 브라우저 로컬 스토리지에 저장됩니다.)

### 2. 코드 모드 🎼
- 메이저 / 마이너 / 세븐(7) / 메이저7 / 마이너7 / 디미니시 / 서스포(sus4) 등
  **49개의 코드 버튼**(7개 루트 × 7개 코드 종류)을 제공합니다.
- 코드 버튼을 누르면 해당 화음이 소리 나고, **피아노에서 눌린 건반이 함께 표시**됩니다.
- **서스테인 페달 버튼**: 켜 두면 누른 음이 일정 시간(약 3.5초) 동안 길게 울려 퍼집니다.
  코드 버튼뿐 아니라 직접 누른 건반에도 적용됩니다.

### 3. 발성 모드 🎤
음계를 한 묶음씩 반복하며 낮은 도부터 높은 도까지 음을 옮겨 가는
**자동 발성 연습** 모드입니다.

- **방향 선택**
  - **오르락내리락**: 낮은 도 → 높은 도로 올라갔다가 다시 내려옵니다. (반복)
  - **오르막길**: 올라가는 방향만 반복합니다.
  - **내리막길**: 내려가는 방향만 반복합니다.
- **음 개수 선택** (한 묶음의 모양)
  - **5음**: 도-레-미-레-도
  - **9음**: 도-레-미-파-솔-파-미-레-도
- **속도 조절**: 슬라이더로 느리게 ↔ 빠르게 조정할 수 있습니다.
- **재생 / 멈춤**: 연주 도중 언제든 멈출 수 있습니다.
- 연주되는 음에 맞춰 피아노 건반과 하단의 음계 가이드가 실시간으로 강조되며,
  가이드는 현재 묶음의 계이름을 보여줍니다.

## 🛠️ 사용 기술 (오픈 소스)

이 앱은 별도의 빌드 과정 없이 **단일 `index.html` 파일**로 동작합니다.

| 라이브러리 / 리소스 | 용도 |
| --- | --- |
| [Tone.js](https://tonejs.github.io/) | 오디오 엔진 — `Tone.Sampler`로 Salamander 그랜드 피아노 샘플 재생, 화음·발성 시퀀스 처리 |
| [Salamander Grand Piano 샘플](https://tonejs.github.io/audio/salamander/) | 실제 피아노 녹음 샘플 음원 |
| [Pretendard GOV](https://github.com/orioncactus/pretendard) | 본문 폰트 |
| [Black Han Sans](https://fonts.google.com/specimen/Black+Han+Sans) | 타이틀 폰트 |
| [Material Icons](https://fonts.google.com/icons) | 아이콘 |
| Web APIs (`MediaRecorder`, `localStorage`) | 연주 녹음 및 저장 |

## 🚀 실행 방법

웹 폰트와 오디오 샘플을 불러오기 위해 **인터넷 연결을 권장**합니다.
피아노 샘플(Salamander) 로딩에 실패하거나 시간이 오래 걸리면, 자동으로
**Tone.js 내장 신디사이저 음색으로 전환**되어 끊김 없이 바로 연주할 수 있습니다.

### 바로 열기
`index.html` 파일을 웹 브라우저로 열면 됩니다.

### 로컬 서버로 실행 (권장)
일부 브라우저는 `file://` 환경에서 오디오 정책이 제한될 수 있어 간단한 로컬 서버 사용을 권장합니다.

```bash
# Python 3
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

### GitHub Pages 배포
저장소 설정의 **Settings → Pages** 에서 브랜치를 선택하면 `index.html`이 그대로 정적 페이지로 배포됩니다.
